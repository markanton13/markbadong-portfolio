# markbadong.com Portfolio

The new portfolio for Mark Anton, positioned around virtual assistance, operations, workflow automation, systems, quality, documentation, and practical web implementation.

## Stack
- Vite
- React
- Custom responsive CSS
- Cloudflare Pages target

## Local development
```bash
npm install
npm run dev
```

## Validation
```bash
npm run lint
npm run build
```

## Cloudflare Pages
- Build command: `npm run build`
- Output directory: `dist`
- Node.js: current supported LTS or newer

## Project status
Phase 1 foundation: homepage architecture, responsive design system, project data model, SEO basics, and deployment-ready static output.
